# Tic Tac Toe

Advanced Tic-Tac-Toe game with Minimax AI and Groq AI integration. Built with FastAPI, React, and Docker following SOLID principles.

## Features

- Single-player Tic-Tac-Toe game
- Dual AI modes: Minimax algorithm and Groq AI
- Modern tech stack: FastAPI + React + Docker
- Professional code quality with PEP 8 compliance
